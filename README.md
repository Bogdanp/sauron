# sauron

![Build Status](https://github.com/racket-tw/sauron/workflows/Racket/badge.svg?branch=master)

WARNING: This project still in early stage.

Sauron is an IDE for Racket language.

### Install and Run

```sh
raco pkg install --auto sauron
racket -l sauron
```

### Develop

```sh
# start IDE
racket main.rkt
# publish document
make publish
```
